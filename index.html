Transistor BJT NPN — Ej. BC547 / 2N2222

Dispositivo semiconductor usado como conmutador y amplificador.

¿Qué es y para qué sirve?

El BJT NPN es un transistor de unión bipolar que permite controlar una corriente mayor entre colector y emisor mediante una corriente más pequeña aplicada a la base. Se usa frecuentemente como conmutador (encendido/apagado) y como amplificador en etapas de señal.

Características clave

  • Terminales: Base (B), Colector (C), Emisor (E).
  • Usos típicos: conmutación digital, preamplificadores, drivers de relés pequeños.
  • Ejemplos comunes: BC547, 2N2222 — ideales para proyectos educativos.

Notas prácticas

Al usarlo como conmutador, se recomienda calcular la resistencia de base para asegurar que el transistor entre en saturación cuando deseado. En montajes reales, proteger con una resistencia y considerar la disipación de potencia.
